# 🌿 Kauppanavigaattori

Kestävän ostamisen sovellus, joka näyttää tuotteiden ympäristövaikutukset.

## Asennus

```bash
pip install -r requirements.txt
```

## Käynnistys

```bash
streamlit run kauppanavigaattori.py
```

## Ominaisuudet

- 🔍 **Tuotehaku** nimellä tai brändillä
- 📷 **Viivakoodihaku** EAN-viivakoodilla
- 🌿 **Eco-Score** (A–E) ympäristövaikutuksesta
- 🥗 **Nutri-Score** ravitsemuksellisesta laadusta
- ⚙️ **NOVA-ryhmä** elintarvikkeen prosessointiasteesta
- 🌍 **Hiilijalanjälki** (CO₂e per 100g)
- 📦 **Pakkausmateriaalit** ja kierrätettävyys
- 🏷️ **Ympäristösertifikaatit** (luomu, Reilu Kauppa jne.)
- 🛒 **Ostoshistoria** – seuraa omia ostoksiasi
- 📈 **Tilastot** – analysoi ympäristövaikutuksesi
